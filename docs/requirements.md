# glance — 要件定義書

## 1. 背景と目的

### 1.1 課題

macOS 標準の Quick Look は `.md` ファイルをプレーンテキストとしてそのまま表示する。Markdown の装飾記法（`#`, `**`, `- [ ]` など）がコードのまま見えるため、内容の把握に時間がかかる。

### 1.2 目的

Finder で `.md` ファイルを選択して Space キーを押すだけで、見出し・太字・リスト・コードブロック・テーブルなどが書式付きで表示される Quick Look 拡張を提供する。

### 1.3 対象ユーザー

- macOS で日常的に Markdown ファイルを扱う開発者・ライター
- エディタを開かずにファイル内容をすばやく確認したいユーザー

---

## 2. 機能要件

### 2.1 Quick Look プレビュー（コア機能）

| ID | 要件 | 優先度 |
|----|------|--------|
| F-01 | Finder で `.md` ファイル選択 → Space で書式付きプレビューを表示 | Must |
| F-02 | 見出し (h1–h6) を適切なフォントサイズ・ウェイトで表示 | Must |
| F-03 | 太字・斜体・取り消し線・インラインコードをレンダリング | Must |
| F-04 | 順序付き/なしリスト、ネストリストを正しくレンダリング | Must |
| F-05 | フェンスドコードブロック（` ``` `）をモノスペースフォント＋背景色で表示 | Must |
| F-06 | テーブルを罫線付きで表示 | Must |
| F-07 | リンクをクリック可能な形式で表示 | Must |
| F-08 | 画像（相対パス）を表示 | Should |
| F-09 | ブロック引用（`>`）を左ボーダー付きで表示 | Must |
| F-10 | 水平線（`---`）をレンダリング | Must |
| F-11 | タスクリスト（`- [ ]` / `- [x]`）をチェックボックスで表示 | Should |

### 2.2 ダークモード対応

| ID | 要件 | 優先度 |
|----|------|--------|
| F-20 | システムのダーク/ライトモードに自動追従 | Must |
| F-21 | テキスト・背景・コードブロック・ボーダーの配色を切り替え | Must |

### 2.3 設定画面（メインアプリ）

| ID | 要件 | 優先度 |
|----|------|--------|
| F-30 | フォントサイズの変更（小 / 中 / 大） | Should |
| F-31 | コンテンツ最大幅の調整 | Nice |
| F-32 | コードブロックの行番号表示トグル | Nice |

### 2.4 メインアプリ

| ID | 要件 | 優先度 |
|----|------|--------|
| F-40 | インストール状態の表示とセットアップガイド | Must |
| F-41 | アプリ内で Markdown ファイルをドラッグ＆ドロップしてプレビュー確認 | Should |

## 3. 非機能要件

| ID | 要件 | 目標値 |
|----|------|--------|
| NF-01 | プレビュー表示速度 | 1MB の .md ファイルを 500ms 以内に表示 |
| NF-02 | メモリ使用量 | プレビュー時 50MB 以下 |
| NF-03 | 対応 OS | macOS 12.0 (Monterey) 以降 |
| NF-04 | サンドボックス準拠 | App Sandbox 対応 |
| NF-05 | ファイルエンコーディング | UTF-8 |

---

## 4. 対応ファイル形式

| 拡張子 | UTType |
|--------|--------|
| `.md` | `net.daringfireball.markdown` |
| `.markdown` | `net.ia.markdown` |
| `.mdown` | `public.markdown` |
| `.mkd` | `text.markdown` |

---

## 5. 制約事項

1. **サンドボックス制限**: Quick Look Extension はプレビュー対象ファイルとその親ディレクトリのみアクセス可能。別ディレクトリの画像参照は表示不可。
2. **予約済みファイルタイプ**: `.txt`, `.html`, `.json` など macOS が予約しているファイルタイプは Quick Look Extension でオーバーライド不可。
3. **JavaScript**: WKWebView 内での任意の JS 実行はサンドボックスで制限される。

## 6. 用語定義

| 用語 | 定義 |
|------|------|
| Quick Look | macOS の Space キーによるファイルプレビュー機能 |
| QL Extension | Quick Look Preview Extension。appex バンドルとして提供 |
| UTType | Uniform Type Identifier。ファイル種別の識別子 |
| cmark | CommonMark 仕様の Markdown パーサー（C 言語実装） |
| App Group | 同一開発者の app と extension 間でデータを共有する仕組み |
